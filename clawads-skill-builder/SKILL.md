# ClawAds Skill Builder

Research, write, and publish a new marketing skill to clawads.io. This skill teaches you how to build skills that any AI assistant can use — OpenClaw, Claude Code, Cursor, or any MCP-compatible tool.

## How to Use

Tell me what marketing topic, guru, or framework you want to turn into a skill. I'll research the framework, write a complete SKILL.md, and guide you through publishing it to clawads.io.

**Just say:** "Build a skill for [topic/guru/framework]"

---

## What Makes a Good Skill

A skill is not a prompt template. A skill is a **framework transfer** — it teaches the AI how an expert thinks, not just what to output.

### The Standard of Quality

A good skill has:
1. **Why this approach is different** — what insight or principle separates this expert's method from generic advice
2. **The actual framework** — the named steps, formulas, or decision process, explained deeply
3. **The psychological mechanism** — WHY each element works
4. **Rules and anti-patterns** — what NOT to do
5. **Ready-to-use output format** — structured output the user can directly use

## Platform Compatibility

Skills work on any AI platform:
- **OpenClaw**: SKILL.md file, install via clawads.io
- **Claude Code**: Same content as CLAUDE.md in project root
- **Cursor**: Add to .cursorrules or CLAUDE.md
- **Any AI assistant**: Paste content as system prompt

**Key principle:** Write skills that teach thinking, not tool commands.

## Output Format

When you invoke this skill to build a new skill:
1. Complete SKILL.md ready to upload
2. The core insight (the ONE thing that makes this framework unique)
3. Quality check against the 5 criteria
4. GitHub upload instructions
