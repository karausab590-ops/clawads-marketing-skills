# ClawAds Marketing Skills

> 24 free AI agent skills for copywriting, ads, email sequences, and growth. Compatible with **Claude Code**, **OpenClaw**, **Codex CLI**, and any tool supporting the [Agent Skills](https://agentskills.io) open standard.

## 🚀 Quick Install

### Claude Code
```bash
# Install all skills
npx skills add karausab590-ops/clawads-marketing-skills

# Or copy individual skills to your project
cp -r alex-hormozi-hook-writer/ ~/.claude/skills/
```

### OpenClaw
Browse and install from [clawads.io](https://clawads.io)

## 📝 Format

All skills follow the [Agent Skills 2.0](https://agentskills.io/specification) open standard:

```
skill-name/
└── SKILL.md    # YAML frontmatter + Markdown instructions
```

Each `SKILL.md` includes:
- **name**: Skill identifier (matches folder name)
- **description**: What it does and when to use it
- **metadata**: Author, version, website
